# Haven v2

This is a reboot of the Haven discord bot.

[![Add to Discord](https://img.shields.io/badge/Add%20to%20Discord-5865F2?style=for-the-badge&logo=discord&logoColor=white)](https://discord.com/oauth2/authorize?client_id=1506465698124267552)

| Environment Variable | Description |
| ----------- | ----------- |
| DISCORD_TOKEN | The token to log into your Discord Application |

| Config | Description |
| ----------- | ----------- |
| prefix | The prefix the bot will listen for when using text commands |
| log_level | The level of logging that should be printed to console |
| error_channel | The discord channel the bot should forward errors to |
